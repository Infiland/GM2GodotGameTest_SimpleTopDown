The purpose of this project is to test GM2Godot capabilities, and to test out full conversion from GameMaker to Godot.

## How to test it

1. Clone this repo
2. Install Godot
3. Make an empty project in Godot
4. Open GM2Godot
5. Select this GameMaker project and the empty Godot project
6. Convert
7. Confirm that the game in Godot is exactly the same as in GameMaker
